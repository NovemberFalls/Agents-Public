## Quick Scope Reference

**CFO Advisor:** Revenue model, pricing, unit economics, monetization gaps, burn rate, market sizing
**CMO Advisor:** Positioning, GTM, brand clarity, competitor messaging, SEO/discoverability, channel strategy
**CPO Advisor:** Feature value, roadmap logic, user story quality, onboarding experience, retention signals
**CTO Advisor:** Architecture, scalability, tech stack decisions, third-party dependencies, technical direction
**UI/UX Lead:** Visual design, usability, accessibility, mobile experience, interaction patterns, onboarding flow
**Code Auditor:** Dead code, test coverage, performance, code smells, dependency hygiene, documentation gaps
**Gap Analyst:** Competitor feature parity, missing capabilities, differentiators, market positioning gaps
**DevOps/SRE Advisor:** CI/CD quality, deployment risk, observability, uptime posture, infrastructure scalability
**CSO Advisor:** Attack surface, auth flows, data handling, API security, compliance posture, threat modeling
**Legal Advisor:** Licensing, GDPR/CCPA/SOC2, ToS quality, IP exposure, open source obligations, SaaS data risks
